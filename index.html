nap

nightmare(less) application platform

Download as .zip Download as .tar.gz View on GitHub

nap is a configurable and extensible set of tools and templates that makes it easier to manage and deploy web applications.

  • Have new apps running in seconds.
  • Get a quick overview of running apps.
  • Update an app in seconds with a single command.
  • Allow developers to have their apps updated automatically when they push their code to the server.

Adding an app

$ nap new hello ruby https://github.com/obfusk/nap-hello.git \
  ruby.server=localhost ruby.port=3001
$ nap bootstrap hello
$ nap start hello
